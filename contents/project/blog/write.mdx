### 1. Github pages에서 Next.js 정적 배포

Github Pages는 jekyll를 이용해 정적 배포됩니다. 그러나 Next.js는 node.js 환경 안에서 빌드되기 때문에, [Github Action을 이용해 빌드 환경을 구축](https://github.com/Seo0H/seo0h.github.io/blob/main/.github/workflows/node.js.yml)했습니다.
